# $\mu$ controller_ligthsAPI
A Dutch API to control LEDs using an API, used for STEM workshop on microcontrollers.

## advancedLighting
A very rudimentary version without API calls. It's just there to get everyting working.

## BasisCode_workshopLichtshow
The basic implementation of the API. Contains a very basic code to show the capabilities of the API calls.

## inspiratie_workshopLichtshow
In this script a more advanced usage of the API is given.
